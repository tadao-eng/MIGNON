# MIGNON LINGO

英語 × インドネシア語 — **90日集中プログラム**。
1日1時間、4つのタスクを毎日潰して、3ヶ月で日常会話レベルを目指す学習アプリ。

## 使い方

`learn/index.html` をブラウザで開くだけ。サーバー不要・スマホ対応。
学習データはブラウザ(localStorage)に自動保存されます。

GitHub Pages 有効化後は `https://<user>.github.io/MIGNON/learn/` でアクセスでき、
スマホで「ホーム画面に追加」するとアプリのように使えます(PWA対応)。

## 1日のプログラム(約60分)

| タスク | 内容 | 目安 |
|---|---|---|
| 単語トレーニング | 間隔反復(SRS)による復習+新規単語 | 約25分 |
| 文法レッスン | 1課(解説+例文+確認クイズ)。英語/インドネシア語を毎日交互に | 約15分 |
| リーディング | 短文読解+設問+全訳確認。文法と逆の言語 | 約10分 |
| 音読・スピーキング | フレーズ10本(EN5+ID5)を音声に続けて発声 | 約10分 |

4タスクすべて完了でその日がクリアとなり、ストリーク(連続記録)が伸びます。

## 収録コンテンツ

- 単語・フレーズ: 英語 約190語 / インドネシア語 約170語(自分で追加も可能)
- 文法レッスン: 各言語15課(計30課) — be動詞〜依頼表現、語順〜yangまで
- リーディング: 各言語12本(計24本) — 自己紹介から3ヶ月後の自分まで
- 90日ロードマップ: 週ごとのテーマ×12週

## 機能

- 間隔反復(SRS) — 1日→3日→7日→14日→30日…と自動で復習間隔を調整
- 音声読み上げ(TTS) — 端末の音声から言語に合うものを自動選択、速度調整可
- ストリーク・XP・レベル・バッジ・週別ロードマップ
- ミニマルUI — 白×ネイビー。学習に集中できる静かな画面

## 注意

学習データは端末のブラウザごとに保存されます。ブラウザのデータを消去すると進捗もリセットされるのでご注意を。
