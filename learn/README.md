# MIGNON LINGO 🇬🇧🇮🇩

英語 × インドネシア語を1年間(365日)かけて毎日コツコツ学ぶためのアプリ。

## 使い方

`learn/index.html` をブラウザで開くだけ。サーバー不要・スマホ対応。
学習データはブラウザ(localStorage)に自動保存されます。

GitHub Pages を有効にすれば `https://<user>.github.io/MIGNON/learn/` でスマホからも使えます。
スマホのブラウザで「ホーム画面に追加」するとアプリのように使えます。

## 機能

- **2言語対応** — 英語(約140語)・インドネシア語(約120語)を内蔵。ミックス学習も可
- **間隔反復(SRS)** — 忘却曲線に合わせて復習カードを自動出題(1日→3日→7日→14日→30日…)
- **1年ロードマップ** — Day 1〜365 のカウントと月ごとの学習テーマ
- **毎日続く仕組み** — 1日の目標・ストリーク(連続記録)🔥・XP・レベル・バッジ
- **楽しく学ぶ** — 4択クイズ・音声読み上げ(TTS)・紙吹雪のお祝い演出
- **単語帳** — カテゴリ別に閲覧、自分の単語も追加可能

## 注意

学習データは端末のブラウザごとに保存されます。ブラウザのデータを消去すると進捗もリセットされるのでご注意を。
